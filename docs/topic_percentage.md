# Topic: Persentase (Percentage)

Students practice converting between percentages, fractions, and decimals.

## 1. Overview
This topic focuses on the relationship between parts of a hundred and other numerical representations.

## 2. Question Types (Alternating)
The generator will alternate between "To Percentage" and "From Percentage" questions.

| Type                | Description                                    | Example                                                   |
| :------------------ | :--------------------------------------------- | :-------------------------------------------------------- |
| **To Percentage**   | Convert a fraction or decimal to a percentage. | $\frac{3}{5} \rightarrow 60\%$ or $0.45 \rightarrow 45\%$ |
| **From Percentage** | Convert a percentage to a fraction or decimal. | $25\% \rightarrow \frac{1}{4}$ or $80\% \rightarrow 0.8$  |

## 3. Implementation Notes
- **No Proficiency Levels**: This topic has a single mode that covers all conversion types.
- **Alternating Logic**: 
    - Even indices ($0, 2, \dots$): "To Percentage"
    - Odd indices ($1, 3, \dots$): "From Percentage"
- **Numbers**: 
    - Fractions will use "friendly" denominators (2, 4, 5, 10, 20, 25, 50).
    - Decimals will have 1 or 2 decimal places.
    - Percentages will be whole numbers or multiples of 0.5 where appropriate.
- **Answer Format**: 
    - For "From Percentage", the answer key should show both simplest fraction and decimal form if possible, or consistent with the question's prompt.
